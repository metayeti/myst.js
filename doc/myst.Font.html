Class: Font

myst.Font(options)

Provides a bitmap font construct. This is a faster alternative to native canvas text rendering routines. It is more consistent since letters and spacings will always be rendered pixel perfect in all browsers, but it is also more limiting since it is confined to a relatively small set of characters. The range of characters is limited to ASCII 32-126.

Class: Tween

myst.Tween(from, to, duration, onUpdateopt, onDoneopt, easefopt, procfopt, resetfopt)

Class used for tweening values.

Constructor

new Tween(from, to, duration, onUpdateopt, onDoneopt, easefopt, procfopt, resetfopt)

Constructs a Tween object.
Parameters:
Name Type Attributes Default Description
from number Value to tween from.
to number Value to tween to.
duration number Duration of tween in milliseconds.
onUpdate tweenUpdateCallback <optional>
 Update callback.
onDone function <optional>
 Triggers when Tween finishes.
easef function <optional>
 myst.ease.linear Easing function. Use one of the functions from myst.ease (myst.ease.linear, myst.ease.quadIn, myst.ease.quadOut, myst.ease.quadInOut, myst.ease.backIn or myst.ease.backOut) or substitute your own.
procf function <optional>
 Value post-process function.
resetf function <optional>
 This function is called whenever the Tween is interrupted through tween.finish() and is intended for resetting an object's state when using multiple, nested tweens.

Examples
// Tween from 0 to 100 over 1.5 seconds with quadInOut easing
(new myst.Tween(0, 100, 1500, function(value) {
   console.log(value);
}, function() {
   console.log('Tween done!');
}, myst.ease.quadInOut)).start();
// Tween from 100 to 0 over 2 seconds with linear easing and floor output values.
(new myst.Tween(100, 0, 2000, function(value) {
   console.log(value);
}, function() {
   console.log('Tween done!');
}, myst.ease.linear, Math.floor)).start();
